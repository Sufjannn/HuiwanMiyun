# 惠玩密云 · 青年玩法 (HuiwanMiyun)

本项目是一个专为密云旅游打造的移动端 H5 沉浸式应用，以“青年玩法”为核心，集成了主题景点探索、互动集章打卡、接驳车预约、青年论坛及景区地图等功能。项目采用流行的玻璃态 (Glassmorphism) UI 设计，提供流畅的交互动画与原生的应用体验。

## 🌟 核心功能

### 1. 四大主题探索
精选密云四大特色景点，按“水、林、云、日”四大主题划分：
- **水主题**：邑仕庄园 · 碧波徽章
- **林主题**：云蒙山 · 森林徽章
- **云主题**：云龙涧 · 云海徽章
- **日主题**：日光山谷 · 日出徽章
*点击首页主题卡片，即可顺滑跳转至对应的详细图文介绍页面（基于秀米外部链接）。*

### 2. 扫码集章闯关系统
- **真实扫码识别**：调用设备摄像头，使用 `jsQR` 库实时解析真实二维码。
- **线性闯关机制**：用户需按照“水 ➔ 林 ➔ 云 ➔ 日”的顺序依次扫码解锁，增加了探索的趣味性和挑战性。
- **状态无缝衔接**：扫码成功后自动跳转阅读景点文章，阅读完毕按“返回”键回到本应用时，系统会自动在屏幕中央弹窗发放刚才解锁的徽章，进度实时同步，惊喜感拉满。

### 3. 预约与论坛互动
- **接驳车预约**：提供多条旅游专线，支持单程/往返票价计算，实时选择出行日期、时间及人数。
- **青年论坛**：支持用户发布图文帖子、点赞互动、发布评论，打造年轻人的旅游分享专属社区。

### 4. 账户与权限管理
- **模拟登录系统**：支持手机号+验证码快捷登录（前端模拟，任意 6 位数字即可）。
- **权限控制**：未登录状态下可浏览主页和景点地图，但预约接驳车、扫码集章、发表论坛帖子及查看个人信息等功能需登录后解锁。未登录时会有“请先登录，点击右上角菜单栏”的贴心指引。
- **个人信息**：支持修改姓名、手机号、身份证号、邮箱等资料。

## 🛠️ 技术实现

- **前端技术栈**：纯 HTML5 + CSS3 + Vanilla JavaScript，单文件组件化结构 (`index.html`)，无复杂框架依赖。
- **UI 风格**：全面引入 Glassmorphism（玻璃态）设计，配合流畅的 `transform` 缩放反馈和渐显动画。
- **数据持久化**：
  - 核心状态（集章进度、用户信息、论坛数据、预约记录）持久化存储于浏览器的 `localStorage` 中。
  - 跨页面跳转状态（如去往秀米外部链接的返回判定、扫码成功标识）利用 `sessionStorage` 进行无缝衔接，解决了返回时重新播放启动动画的问题。
- **第三方库**：仅引入 `jsQR.min.js` 用于纯前端的摄像头二维码图像解析。

## 🚀 运行与测试说明

1. **直接运行**：本项目为纯静态前端页面，直接在任意现代浏览器中双击打开 `index.html` 即可运行。
2. **最佳体验**：强烈建议在浏览器开发者工具 (F12) 中开启 **移动端设备模拟 (Device Toggle)**，或直接使用手机浏览器访问以获得最佳触控与视觉体验。
3. **测试集章**：
   - 点击右上角菜单 (☰) 进行登录。
   - 准备好对应主题的秀米文章链接生成的二维码。
   - 点击“📷 扫码集章”，允许浏览器使用摄像头进行扫描。
   - 请务必按照“水、林、云、日”的顺序逐一解锁。

---
*注：本项目为展示级高保真原型，无后端数据库支持，所有数据均保存在本地浏览器中（设备级存储）。因此同一设备/浏览器共享相同的集章进度，清除浏览器缓存会导致所有数据重置。*
